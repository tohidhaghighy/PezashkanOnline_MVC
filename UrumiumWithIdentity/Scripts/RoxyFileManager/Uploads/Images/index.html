Welcome to CKEditor SDK!

CKEditor SDK is a collection of resources designed to help you get familiar with all editor features and make the process of implementing, configuring and integrating CKEditor 4 with your website or application easier. It contains the following parts:

  • Samples
    Dozens of live demos showcasing numerous CKEditor 4 aspects and features, with source code ready to copy and implement in your own solution. You can browse samples in the sidebar on the left.
  • Documentation
    Plenty of articles and tutorials that describe everything a CKEditor 4 developer needs to know, from the installation process through the best practices and functionality overview to information on creating your own plugins.
  • CKEditor API
    The entire CKEditor 4 API documentation — all methods, events, properties and configuration options explained in one place.

CKEditor 4 Samples

All editor features were divided into two main categories, available in the sidebar on the left:

  • End-user Features – functionality that your users will see and use.
  • Integration Features – more advanced stuff that a CKEditor 4 integrator might need.

The samples are complemented with relevant documentation articles which explain the features even more broadly — these are always available in the Documentation link next to the sample header. You can also find additional reading in the Related Features section down below the sample.

Most importantly, though, you can view and copy the exact source code of each editor demo in the Get Sample Source Code section at the bottom of the sample pages. This code uses the CKEditor CDN, so you can save it and run locally without the need to download CKEditor 4 itself!

Note: CKEditor SDK is being expanded all the time. Any bugs and tips should be reported here. Thanks!
